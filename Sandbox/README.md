# Sandbox
Simple JavaScript project to test things.

## Prerequisites
You will need:
- [npm](https://www.npmjs.com/) installed:
```
brew install npm
```

## Steps
On your local machine, you need to first install all dependencies:
```
npm install
```

To run the sandbox, simply run:
```
npm run start
```
